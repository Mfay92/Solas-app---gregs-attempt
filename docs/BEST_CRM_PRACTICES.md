## 2. Essential Features (The "Must-Haves")

### A. The Property DNA üè†
Don't just store an address. A property profile should tell a story:
- **Compliance Traffic Lights:** Red/Amber/Green status for Gas, Electric, Fire, Legionella. *If it's red, the system should scream.*
- **Void Management:** Every day empty is money lost. Show "Days Void" and "Revenue Lost" prominently.
- **HMO Licensing:** Track license capacity vs. actual occupancy.

### B. The Person-Centred Core üë§
For Supported Living, the "Tenant" is a "Service User".
- **Risk Assessments:** Digital, version-controlled, and easily accessible.
- **Support Hours:** Track commissioned hours vs. delivered hours.
- **Incident Reporting:** A streamlined flow to log incidents (falls, behavioral) that auto-notifies relevant parties.
